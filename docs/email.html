Email

GitHub Pages only serves static files—it cannot send SMTP mail. This page helps you open your mail app with a prefilled message, and builds a GitHub CLI command to run the repo’s Send email (SMTP) workflow (after you add secrets once).

1. Mail app (mailto)

Uses your device’s default email client. Nothing is sent from GitHub.

2. GitHub Actions (SMTP)

One-time: in the repo on GitHub, add these Actions secrets (Settings → Secrets and variables → Actions): MAIL_SMTP_HOST, MAIL_SMTP_USERNAME, MAIL_SMTP_PASSWORD, MAIL_FROM (e.g. Your Name <you@domain.com>). For Gmail, use an App Password and host smtp.gmail.com, port 465.
