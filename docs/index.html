What this is

Random-stuff is exactly what it sounds like: a single place to collect random projects I’m tinkering with—scripts, tiny apps, web experiments, notes, whatever I’m iterating on when the mood strikes.

Vibe coding, for me, means following the energy: trying an API, sketching a UI, refactoring because it feels good, or shipping something scrappy just to see it live. Not every folder or commit will be “production ready,” and that’s intentional.

You’re reading the public face of the docs/ folder; the rest of the repo is whatever those projects need. When something grows up, it might graduate to its own repository—or it might stay right here.

How this page gets online

The site is static HTML on GitHub Pages, deployed with GitHub Actions whenever I push to main. Source for this page lives under docs/.
