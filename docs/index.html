What lives here

The repository is a scratch space: ideas that do not need their own project yet, or content you want versioned and easy to share. The live site you are reading is the public face of that folder—no build step, just HTML and assets checked into docs/.

If you add more pages later (for example docs/other.html), link them from here or from a simple navigation block so visitors can find them.

How publishing works

GitHub Pages is configured to use GitHub Actions. The workflow Deploy GitHub Pages uploads the contents of docs/ as the site artifact and deploys it to the github-pages environment.

  • Trigger: push to the main branch (or run the workflow manually).
  • Source: files under docs/ in this repository.
  • URL: neutro74.github.io/Random-stuff

Avoid adding a second workflow that deploys the repository root unless you add a root index.html; otherwise the site root can end up empty and return 404.

Ideas for next steps
